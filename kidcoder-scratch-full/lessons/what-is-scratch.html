Meet Scratch

Scratch is a free tool for making stories, animations, and games using code blocks. You snap blocks together to control characters (sprites) on a stage.

  • Sprites: characters and objects.
  • Backdrops: the background scene.
  • Blocks: the code you snap together.
  • Run: click the green flag to start.
