What is Scratch?

Scratch is a free tool that lets you make stories, games, and animations by snapping code blocks together just like for example LEGO.

  • Stage: where your game plays.
  • Sprites: characters and things you control.
  • Blocks: pieces of written code "compressed" into a block that tells sprites what to do.
