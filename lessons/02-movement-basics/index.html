Movement Basics

Motion blocks move a sprite on the stage. You’ll use move, turn, go to, glide, and if on edge, bounce.

  • Stage center: (0, 0). Right is +x, up is +y.
  • Facing direction affects move.
  • go to x: y: teleports; glide animates.

Arrow Key Movement (X/Y)

when up arrow key pressed
  change y by 10
when down arrow key pressed
  change y by -10
when right arrow key pressed
  change x by 10
when left arrow key pressed
  change x by -10

Directly edits coordinates for crisp grid-based movement.

Arrow Key Movement (Facing + Move)

when right arrow key pressed
  point in direction 90
  move 10 steps
when left arrow key pressed
  point in direction -90
  move 10 steps
when up arrow key pressed
  point in direction 0
  move 10 steps
when down arrow key pressed
  point in direction 180
  move 10 steps

Uses facing to keep animations consistent with motion.

Snap to Start

when green flag clicked
  go to x: 0 y: -100
  point in direction 90

Always reset position & direction at the start of the game.

Smooth Movement (Glide)

when this sprite clicked
  glide 0.5 secs to x: 0 y: 0
  glide 0.5 secs to x: 100 y: 100

Glide animates movement over time.

Bounce at Edges

when green flag clicked
  forever
    move 7 steps
    if on edge, bounce

Great for AI balls or roaming enemies.

Screen Wrap (Optional)

forever
  move 8 steps
  if < x position > 240 then set x to -240
  if < x position < -240 then set x to 240
  if < y position > 180 then set y to -180
  if < y position < -180 then set y to 180

Wrap-around movement like classic arcade games.

Jump (Simple)

when space key pressed
  repeat 10
    change y by 5
  repeat 10
    change y by -5

Basic hop. (Gravity physics comes later.)

Follow the Mouse

when green flag clicked
  forever
    point towards [mouse-pointer v]
    move 6 steps

Great for chasing gameplay.

Challenge

  • Add diagonal movement (pressing two keys).
  • Make a patrol path with glide and wait.
  • Create a start position hotkey: press R to reset.
