Loops

Run code many times with repeat, repeat until, and forever.

  • Shorter scripts, fewer bugs.
  • Great for animation and AI.
  • Use wait to control speed.

Repeat N Times

repeat (10)
  move 10 steps
end

Equivalent to 10 ‚Äúmove 10‚Äù blocks, but cleaner.

Forever Loop

when green flag clicked
  forever
    if on edge, bounce
    move 6 steps
  end

Background behaviors that never stop until the game ends.

Repeat Until

repeat until < touching [Goal v] ? >
  move 5 steps
end
say [Made it!] for 1 seconds

Stop looping when the condition becomes true.

Nested Loops (Square)

repeat (4)
  repeat (10)
    move 5 steps
  end
  turn 90 degrees
end

Patterns and shapes are easy with nested loops.

Animation Loop

when green flag clicked
  forever
    next costume
    wait 0.1 seconds
  end

Simple idle/walk animations for any sprite.

Spawn Over Time

when green flag clicked
  forever
    create clone of [Enemy v]
    wait (2) seconds
  end

Periodic spawns using forever + wait.

Timer with Loop

when green flag clicked
  set [TimeLeft v] to (60)
  repeat until < (TimeLeft) = (0) >
    wait (1) seconds
    change [TimeLeft v] by (-1)
  end
  broadcast [time-up v]

Classic countdown using repeat-until & waits.

AI Patrol

when green flag clicked

    glide (1) secs to x: (150) y: (-50)
    glide (1) secs to x: (-150) y: (-50)

Looping patrol motion for enemies or platforms.
