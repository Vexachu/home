Meet Python!

Our coding language

  • Just as many speaking languages exist (English, Spanish, Chinese…), so do coding languages (Python, Java, JavaScript, C++).
  • Python is one of the most popular and friendly for beginners.
  • Its code looks a lot like plain English — that’s why we start here.

I won’t get into more code just yet, but here’s an example:

print("Hello, world!")
  • The computer will print what we tell it to.
  • You can almost read it like a sentence.
  • Instead of “say”, we use the word print.

But why code…

Become a creator, not just a player!

  • Programming lets you build your own games, apps, and stories.
  • It helps you think logically and solve problems.
  • You get to invent new and amazing things!

You can make many kinds of games. Popular ones like Minecraft use programming — especially Python, which we’ll explore soon.

It’s not just games though… you can use programming for Instagram, movie editing, or even controlling robots.

But what is code?

Code is like a recipe!

  • To make a cake, you need a recipe. To make a computer do something, you need code.
  • If you forget a step in a recipe, the cake won’t turn out right.
  • If you forget a step in code, the computer gets confused!

Computers and Code

How to talk to a computer

  • Computers are smart… but not that smart.
  • They need step-by-step instructions for everything they do.
  • Just like a pet needs training, a computer needs instructions — this is “code.”
