Meet Programming!

Our coding languages

  • Just as many speaking languages exist (English, Spanish, Chinese…), so do coding languages (Python, Java, JavaScript, C++).

I won’t get into more code just yet, but here’s an example of code in Python, a language that looks a lot like plain English:

print("Hello, world!")
  • The computer will print what we tell it to.
  • You can almost read it like a sentence.
  • Instead of “say”, we use the word print.

But why code…

Become a creator, not just a player!

  • Programming lets you build your own games, apps, and stories.
  • It helps you think logically and solve problems.
  • You get to invent new and amazing things!

You can make many kinds of games. It's not just popular games like Minecraft that use programming, but every single game!

It’s not just games though… you can use programming to make any sort of app or websit or even use it to control robots.

But what is code?

Think of code as the instructions for a computer, like a recipe for baking a cake.

  • Just as a recipe guides you to make a delicious cake, code guides a computer to perform tasks.
  • Missing a step in a recipe can ruin the cake. Similarly, missing or messing up a step in code can cause the computer to get confused or not work as expected.
  • Basically, code is the set of instructions that helps computers understand what to do!

Computers and Code

How to talk to a computer

  • Computers are smart… but not that smart.
  • Computers don't actually do anything themselves!
  • They need step-by-step instructions for everything they do.
  • Just like a pet needs training, a computer needs instructions, and that is what we call “code.”
