  forever
    if < touching color [#2b2b2b] ? > then
      move -6 steps
    end

Damage with Invincibility

when I receive [player-hit v]
  if < (Invincible) = [0] > then
    change [Lives v] by -1
    set [Invincible v] to 1
    wait 1.5 seconds
    set [Invincible v] to 0
  end

Prevent instant double-hits by using an invincibility window.

Goal & Win

forever
  if < touching [Goal v] ? > then
    broadcast [win v]
  end

when I receive [win v]
  stop all

Simple win condition. Replace stop all with a victory scene if you want.
