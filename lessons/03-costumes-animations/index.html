Costumes & Animations

Costumes are frames. Switching them creates animation.

  • Use next costume or switch costume to.
  • Add short wait for smoother motion.
  • Pair with movement for walking.

Walk Cycle (Two Frames)

when right arrow key pressed
  point in direction 90
  switch costume to [walk1 v]
  move 8 steps
  wait 0.05 seconds
  switch costume to [walk2 v]
  move 8 steps

Alternate frames as you move.

Idle Animation

when green flag clicked
  forever
    switch costume to [idle1 v]
    wait 0.2 seconds
    switch costume to [idle2 v]
    wait 0.2 seconds

Subtle breathing/blink while standing.

Flip to Face Direction

when right arrow key pressed
  set rotation style [left-right v]
  point in direction 90
  move 8 steps
when left arrow key pressed
  set rotation style [left-right v]
  point in direction -90
  move 8 steps

Automatically mirrors sprite art.

Attack / Action Animation

when space key pressed
  broadcast [attack v]

when I receive [attack v]
  repeat 3
    next costume
    wait 0.05 seconds
  switch costume to [idle1 v]

Use a broadcast to keep action self-contained.

Animated Teleport

when this sprite clicked
  repeat 4
    change effect [ghost v] by 25
    next costume
    wait 0.05 seconds
  go to x: (pick random -200 to 200) y: (pick random -150 to 150)
  set effect [ghost v] to 0

Fade + jump for a clean teleport.

Hit Flash

when I receive [hit v]
  repeat 5
    set effect [color v] to 50
    wait 0.03 seconds
    clear graphic effects
    wait 0.03 seconds

Quick feedback when damaged.

Footstep Sync

when right arrow key pressed
  start sound [step v]
  next costume
  move 8 steps

Pair costume changes with audio cues.

Backdrops for Scenes

when I receive [next-scene v]
  switch backdrop to [Level 2 v]
  
when backdrop switches to [Level 2 v]
  broadcast [spawn-enemies v]

Hook scene change to spawns and logic.
