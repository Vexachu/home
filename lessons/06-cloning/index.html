Bullet Clones

when [space v] key pressed
  create clone of [Bullet v]

when I start as a clone // in Bullet
  go to [Player v]
  point in direction (direction of [Player v])
  repeat until < touching [edge v] ? >
    move (14) steps
    if < touching [Enemy v] ? > then
      broadcast [enemy-hit v]
      delete this clone
    end
  end
  delete this clone

Player shoots bullet clones that clean themselves up.

Limit the Clone Count

  set [CloneCount v] to (0)

when I start as a clone
  change [CloneCount v] by (1)
  // ... do things ...
  delete this clone

when I receive [cleanup v]
  delete this clone

// In a controller:
forever
  if < (CloneCount) > (50) > then
    broadcast [cleanup v]
  end

Keep memory safe by limiting clones and cleaning up.

Wave Spawner

when green flag clicked
  set [Wave v] to (1)
  forever
    repeat (Wave)
      create clone of [Enemy v]
      wait (0.2) seconds
    end
    change [Wave v] by (1)
    wait (2) seconds
  end

Each wave spawns more enemies than the last.
