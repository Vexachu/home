Lesson 1: Intro to Turtle

You can think of Python turtle like a little pet turtle you can move with code. Today we’ll move it forward!

Each turtle needs a name. For example, mine is bob.

import turtle
bob = turtle.Turtle()
bob.forward(50)

This makes bob move forward 50 steps and leave a trail behind!

1 / 10

Setup: import turtle + make a turtle

Two lines you’ll use in almost every turtle program:

import turtle                # brings the turtle library into your code
bob = turtle.Turtle()        # creates a turtle object named bob
  • import loads extra features (a library).
  • turtle.Turtle() constructs a new turtle you can control.

Turning the Turtle

        
You can use .left() or .right() to make your turtle turn!

        
import turtle
bob = turtle.Turtle()
bob.forward(80)
bob.left(90)
# the code above makes bob turn left by 90 degrees
bob.forward(80)
bob.left(90)
bob.forward(80)
bob.left(90)
bob.forward(80)

        
This makes a square! Try changing the distance or angles to make new shapes.

Pen Up & Pen Down

        
penup() lifts the pen (no drawing). pendown() puts it back down (draws again).

        
import turtle
bob = turtle.Turtle()

bob.forward(60)     # draw
bob.penup()         # stop drawing
bob.forward(60)     # move without a line
bob.pendown()       # draw again
bob.left(90)
bob.forward(60)

Color

        
Use color() with a name ("white", "red") or a hex like "#00ff88".

        
import turtle
bob = turtle.Turtle()

bob.color("white")     # line + fill color
# you can also do: bob.color("yellow") or bob.color("#4ade80")

Line Thickness

        
Change the line width with pensize() (sometimes called width()).

        
import turtle
bob = turtle.Turtle()

bob.pensize(5)      # thicker line
bob.forward(80)

# Optional: change only the turtle icon size (not the line)
# bob.shapesize(1.5)

Regular Shapes: the trick

        
For an n-sided regular polygon, turn by 360 / n each step and repeat n times.

        

          

            
import turtle
bob = turtle.Turtle()

n = 6          # try 3 (triangle), 4 (square), 5, 6, 8...
step = 70
angle = 360 / n
for _ in range(n):
    bob.forward(step)
    bob.left(angle)

            
Idea map: triangle → 120°, square → 90°, pentagon → 72°, hexagon → 60°.
