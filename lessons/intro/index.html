Turning the Turtle

    
You can use .left() or .right() to make your turtle turn!

    
import turtle
bob = turtle.Turtle()
bob.forward(80)
bob.left(90) 
#the code above makes bob turn left by 90 degrees
bob.forward(80)
bob.left(90) 
bob.forward(80)
bob.left(90) 
bob.forward(80)
